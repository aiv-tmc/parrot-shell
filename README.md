## Install (Linux)
1. Clone the repository:
`$ git clone https://github.com/aiv-tmc/parrot-shell.git`

2. Go to the directory with the code:
`$ cd ~/Download/parrot-shell`

3. Start making the program:
`$ sudo make all`
